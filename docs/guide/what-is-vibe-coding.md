# 什么是 Vibe Coding

## 本节目标

- 理解 Vibe Coding 的核心概念
- 了解传统编程与 Vibe Coding 的区别
- 明确本教程的学习目标和项目规划

## 一句话理解

Vibe Coding（氛围编程）就是**用自然语言跟 AI 对话，让 AI 帮你写代码**。你不需要学编程语言，只需要说清楚你想要什么。

## 传统编程 vs Vibe Coding

传统编程是这样的：你需要学会一门编程语言（比如 JavaScript、Python），然后一行一行地写代码，告诉电脑该做什么。这个过程门槛很高，学习周期很长。

Vibe Coding 是这样的：你打开一个 AI 编程助手（我们用 Cursor），用中文（或任何语言）描述你想做的东西，AI 帮你把代码写好。你只需要确认结果是不是你想要的。

**打个比方**：传统编程像是自己盖房子，你得会砌砖、接水电。Vibe Coding 像是跟一个建筑师沟通，你说"我想要一个两室一厅，客厅要大一点"，建筑师帮你搞定。

## 你需要学会的核心技能

在这个教程中，你会学到三个核心技能：

1. **描述需求**：怎么跟 AI 说清楚你想要什么
2. **验证结果**：怎么判断 AI 做出来的东西对不对
3. **解决问题**：出了问题怎么让 AI 帮你修

就这三件事。不需要学编程语言，不需要看懂代码。

## 我们要做什么

这个教程会带你完成两个真实项目：

- **番茄时钟**：一个帮助你专注工作的计时应用，25 分钟工作 + 5 分钟休息
- **贪吃蛇**：经典的贪吃蛇游戏，还会加上排行榜等功能

第一个项目很简单，你只需要跟 AI 聊天就能完成。第二个项目会引入一些进阶工具，让你体验更强大的 AI 开发方式。
