# 常见问题

## 环境相关

### `npm run dev` 后浏览器打不开页面

告诉 AI：

> 运行 npm run dev 之后终端显示了地址，但浏览器打开后是空白页面，请帮我排查

### 修改代码后页面没有自动刷新

告诉 AI：

> 我修改了代码，但浏览器没有自动更新，需要手动刷新才能看到变化。请检查热更新是否正常工作

### 终端里出现红色报错信息

不要慌。把报错信息复制下来，发给 AI：

> 终端里出现了以下报错信息，请帮我解决：[粘贴报错信息]

## 游戏功能相关

### 蛇的移动方向能反向（比如向右走时直接向左）

告诉 AI：

> 蛇正在向右移动时，按左方向键蛇会直接掉头往左走，这样会直接撞到自己。请禁止蛇 180 度掉头

### 游戏在不同屏幕尺寸下表现不一致

告诉 AI：

> 游戏在我的大屏幕上太小了，能不能让游戏区域根据屏幕大小自适应？

### 在线排行榜加载慢或者加载失败

告诉 AI：

> 排行榜数据加载很慢（或者加载失败），请检查 Supabase 的连接配置是否正确，并加上加载中的提示

## MCP 相关

### MCP 连接不上

告诉 AI：

> MCP server 连接失败，请帮我检查配置是否正确

如果 AI 也解决不了，尝试重启 Cursor。

### 不确定 MCP 是否在工作

告诉 AI：

> 我不确定 MCP 是否正常工作，能不能做一个简单的测试来确认？

## Agent Skills 相关

### 不知道该安装哪些 Skills

告诉 AI：

> 我在做一个 JavaScript 网页游戏项目，推荐一些适合的 Agent Skills

### 安装 Skill 后想卸载

告诉 AI：

> 帮我卸载 [Skill 名称] 这个 Agent Skill

## 部署相关

### 想把游戏部署到网上

告诉 AI：

> 我想把这个贪吃蛇游戏部署到网上，让别人也能玩。推荐一个最简单的免费部署方式，帮我完成部署

AI 通常会推荐 Vercel 或 Netlify 等免费的静态网站托管服务。

### 部署后在线排行榜不工作

告诉 AI：

> 游戏部署到网上后，在线排行榜功能不正常了。在本地开发时是正常的。请帮我检查是不是环境变量或者跨域的问题
