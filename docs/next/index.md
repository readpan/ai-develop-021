# 下一步

恭喜你完成了两个项目！你已经掌握了 Vibe Coding 的核心技能。

## 你现在能做什么

经过番茄时钟和贪吃蛇两个项目的练习，你现在可以用 Vibe Coding 来做各种各样的东西。以下是一些方向：

### 工具类应用
- 记账本 / 消费追踪
- 习惯打卡器
- 密码生成器
- Markdown 笔记应用

### 游戏
- 扫雷
- 2048
- 俄罗斯方块
- 记忆翻牌游戏

### 个人网站
- 个人简历 / 作品集
- 博客
- 相册网站

### 更进阶的应用
- 带用户登录的应用
- 能跟数据库交互的全栈应用
- 移动端 App（使用跨平台框架）

## 几个建议

1. **从你真正需要的东西开始**。做一个你自己会用的工具，比做一个练习项目有趣得多。

2. **先做最简单的版本**。不要一上来就想做一个很复杂的应用。先让核心功能跑起来，再慢慢加功能。

3. **遇到问题别怕**。你已经学会了最重要的技能——把问题描述给 AI，让 AI 帮你解决。

4. **探索更多 MCP 和 Skills**。随着你做的东西越来越复杂，你会需要更多的工具。用你学到的方法去搜索和安装它们。

## 持续学习

Vibe Coding 的工具和方法还在快速发展。保持好奇心，关注 Cursor 和 AI 编程领域的新动态。

你今天学会的核心方法——**用自然语言描述需求，让 AI 帮你实现**——这个思路不会变。工具会升级，但方法是一样的。
